How to add a page?

To add a page, you need to type it's title and content in the respective boxes. It's just simple if you use following tools while adding your content;

  • We have heading of six size.
    One # before something will make it the biggest heading. For example, # Heading would give;

    Heading

    ## Heading

    Heading

    ### Heading

    Heading

    ..............
    ..... so 6 # will give us the smallest heading :
    ###### Heading
    Heading

  • Use **NAME** for bold text. It will give: NAME

  • Use [NAME OF LINK](URL OF LINK) for links.
    For example, a link to this page will be like; [Guide](url of this page) and will look like Guide

Voila! You are ready to add pages.
